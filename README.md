# 🏆 KBC Quiz Game in Python

This is a console-based **Kaun Banega Crorepati (KBC)** game developed in Python. The project is designed to practice core Python programming concepts such as functions, loops, conditional statements, lists, dictionaries, and user input handling.

## 🚀 Features
- 🎯 Multiple-choice quiz questions
- 💰 Prize money tracking
- ✅ Answer validation
- 📊 Score calculation
- 🎮 Interactive console interface
- 🔁 Easy to add new questions

## 🛠️ Technologies Used
- Python 3

## 📚 Concepts Covered
- Functions
- Loops
- Conditional Statements
- Lists & Dictionaries
- Input/Output
- Basic Error Handling

## ▶️ How to Run
```bash
python KBC.py
